# プチ約束
![プチ約束ロゴ](https://i.gyazo.com/ca479984e1518fb8cdd6f65df8a05eec.jpg)

## 概要
プチ約束は日常の小さな口約束をLINE上で管理できるサービスです。
あの時のお金の貸し借りや、その時のご飯の約束など、思いついた時にLINEの公式アプリから約束を作成することができます。
![ペルソナ](https://i.gyazo.com/ec064d18469bd093ae70085c4f8a479d.png)

1. LINEの公式アカウントから、アプリを開きます。
2. 約束の内容を入力します。
	![約束のプチり方](https://i.gyazo.com/ace836e99e105f39c73ffe61e1908ced.png)
	![約束の種類](https://i.gyazo.com/525bf717df426c4df86bf41b032a98dd.png)
3. アプリ上でLINEの友達に約束が送信されます。
4. 相手が受け取ったら約束が成立します。

![約束のプチり方](https://i.gyazo.com/320b4ceda02f062e38822f8412ea3722.jpg)

一度約束が成立すると、アプリがランダムにリマインドを送信してくれます。
手動でリマインドを送っても、アプリからの通知に紛れてくれます。

![約束をプチった後](https://i.gyazo.com/95c2a4f95f219a55b875f3138b8b3fdd.jpg)

## 開発期間
2024-03 ~ 2024-04（2週間）

## 使用した技術
- フロントエンド: TypeScript, React, Next.js, Yamada UI, Jotai, GraphQL, LINE FrontEnd Framework, Zod
- バックエンド: TypeScript, prisma, hono.js, Apollo server
- その他: Python, Next.js, Supabase, Deno, Vercel, Google Cloud Scheduler

※バックエンドはNext.js Api Routeを使用しています。
