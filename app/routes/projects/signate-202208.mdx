# SIGNATE Student Cup 2022【

## 概要

SIGNATE の AI 開発コンペティションに研究室の同僚と参加した際の成果物があります。
機械学習を用いた自然言語処理モデルの精度を競うもので、私はチームのリーダーを務めつつ、データ分析、前処理、モデル構築を担当しました。
また、大学の GPU サーバーを使用するにあたり、チームメンバが Docker を用いて環境構築できるようにマニュアルを作成し、SSH 接続から実際に動かせるようになるところまでフォローしました。
残念ながら実施期間が院試と被っていたこともあり、あまりよい結果にはなりませんでしたが、銅メダルを獲得できました。
以下がコンペの結果の URL で、63 位が自分のチームです。
