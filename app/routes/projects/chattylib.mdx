# Chatty Library

## 概要

[NPO 法人サイエンス・アクセシビリティ・ネット](https://saccessnet.com/)で Chatty Library の開発に 2 年間携わりました。
Chatty Library とは、視覚障害のある方に向けて、本や教科書の内容を読み上げる Web アプリです。
私は Chatty Library がサポートする教科書や論文などのコンテンツを増やすために、GPU サーバと画像認識を用いた認識処理サービスの開発を行いました。

## 業務内容：

- 数式や第二水準漢字を含む PDF をテキストに変換するシステムのバックエンド部分の開発
- 物理 GPU サーバ(Ubuntu)4 台のセットアップ、環境構築
- AWS の S3 サーバーにアップロードされた画像を GPU サーバー 4 台で監視＆推論
  - 推論結果を S3 サーバーにアップロード
  - エラーハンドリングやロギングの開発
- フォント推定のモデル開発
- Tesseract を用いた文字認識の MLOps の設計
- Transfomer を用いた文章の読み順推定

## 期間

2023-4 ~ 2025-4
